Gra tworzona w silniku GODOT. 

Niektóre założenia: 

-Izometryczna gra fabularna

-Gra na jednego gracza

-Rozwinięty system dialogowy

-System ekwipunku

-System walki

-System zapisywania i wczytywania save'ów

-Rozgrywka na 2/3 godziny
